# Project Rules & Design Preferences

## Design Principles
- **Theme**: Official Discord Dark Client theme.
  - Background Primary: `#313338` (main page sections)
  - Background Secondary: `#2b2d31` (cards, containers)
  - Background Tertiary/Darkest: `#1e1f22` (navbars, footers, headers)
  - Accent Color (Blurple): `#5865f2` (buttons, call-to-actions, active state)
  - Success Color (Green): `#248046` or `#23a55a` (download actions)
  - Text Primary: `#dbdee1`
  - Text Muted: `#949ba4`
- **Layout & Visuals**:
  - Web applications must look premium, modern, and highly interactive.
  - Avoid generic web layouts. Use custom-tailored elements mimicking Discord client features.
  - Screenshot of the app (`/public/image.png`) must be showcased inside a 3D Floating Mockup with perspective tilt/hover effects.
  - Feature cards must contain visual charts demonstrating Rust performance (RAM/CPU) vs. Electron.
  - Incorporate a simulated interactive code editor showcasing the ease of configuring custom JSON rules.
