Zero Configuração

Vem com suporte nativo pré-configurado para mais de 20 dos aplicativos mais populares como VS Code, Spotify, Steam, navegadores e pacote Adobe.

Gráficos & Botões

Adicione imagens grandes e pequenas, contadores de tempo decorrido e botões de ação clicáveis diretamente no seu status sem pagar pelo Nitro.

Motor Inteligente

🚀

Extremamente Leve

Construído em Tauri + Rust, ele consome quase zero de memória RAM e CPU, rodando silenciosamente no segundo plano do seu sistema.

Para Desenvolvedores

Ajude-nos a Expandir a Biblioteca

O Better RPC é mantido pela comunidade. Se você quer adicionar suporte para um aplicativo que você usa, basta criar um mapeamento JSON rápido e declarativo.

✔ Mapeamento de processos via Expressões Regulares (Regex)
✔ Interface moderna em TypeScript + React
✔ Core ultrarrápido programado em Rust
⭐ Contribuir no GitHub
// Exemplo de regra JSON customizada: 
{
  "name": "My Custom Editor",
  "processName": "my-editor",
  "rules": [
    {
      "windowTitleRegex": "^Editing - (.*)",
      "details": "Desenvolvendo código",
      "state": "Arquivo: $1"
    }
  ]
}
