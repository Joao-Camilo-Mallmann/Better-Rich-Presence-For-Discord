How it works

Install it. Open it. Done.

No developer portal, no tokens, no configuration files. Better RPC reads your running processes and maps them to Discord Rich Presence.

⚡

Automatic detection

Recognizes VS Code, Spotify, games, browsers, and any desktop program without configuration.

🦀

Native Rust performance

Built with Tauri. Under 20 MB RAM, near-zero CPU. No Electron overhead.

🔧

Fully customizable

Edit rules to control what shows on your profile. Add custom app names, icons, and detail text.

📦

Open source

MIT licensed. Read the code, fork it, contribute. No tracking, no accounts, no data collection.
