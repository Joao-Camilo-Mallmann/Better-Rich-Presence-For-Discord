FAQ

Frequently Asked Questions

How does Better RPC detect applications?

Better RPC automatically tracks active windows in the background using native APIs (Win32 on Windows). When you switch windows, it immediately matches the executable name or window title to update your Discord status with zero manual input.

Can I add any custom app to Discord status?

Yes! Better RPC lets you easily add custom applications. Simply type the name of the executable or window title, set your custom status text and active icons, and it will start showing up on your profile instantly.

Is it really lightweight?

Unlike other Electron-based custom RPC applications that use hundreds of megabytes of RAM, Better RPC is written in Rust and Tauri. It uses less than 20MB of memory and practically 0% CPU, making it the most lightweight Discord RPC app available.

Why is it the easiest Discord Rich Presence tool?

Most Discord status customizers require creating a developer application, setting up tokens, or writing configuration files. Better RPC handles everything inside a simple desktop app. It's completely plug-and-play!
